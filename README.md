# 🚀 Velora AI

Velora is a simple AI-powered productivity assistant built with Python and Streamlit.

## ✨ Features
- AI Chat (simple version)
- Study timer
- Session tracking
- Lightweight design

## 🛠 Tech Stack
- Python
- Streamlit
- SQLite

## ▶️ Run Project
streamlit run app.py

## 📌 Purpose
This project is built for learning and portfolio purposes.# 🚀 Velora AI
